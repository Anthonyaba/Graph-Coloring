3.目前禁忌算法500.5平均速率约为17w次每秒
4.在混合进化算法中，生成初始解个数的多少，在实验下，我认为应与对新生成的解S0进行的禁忌搜索的迭代步数相关联，在初始解个数多的情况下，可设置较少的迭代步长，达到对种群快速进化的目的，即发挥数量的优势以数量取胜。而在初始解个数少的情况下，则设置较长迭代步长，使优质解更优（即冲突数少），发挥质量优异的优势。
